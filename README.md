
# 📊 Virat Kohli Career Statistics Dashboard (2008–2025)

## 📌 Overview  
This project is an interactive **Power BI dashboard** that analyzes the cricket career of Virat Kohli across ODI, Test, and T20I formats.  

It provides insights into performance trends, consistency, and scoring patterns using dynamic visualizations.

---

## 🎯 Objectives  
- Analyze performance from **2008 to 2025**  
- Compare stats across formats (ODI, Test, T20I)  
- Identify peak performance years  
- Visualize consistency using 50s and 100s  

---

## 🛠️ Tech Stack  
- Power BI Desktop  
- Excel / CSV Dataset  
- Power Query (ETL)  
- DAX (Data Analysis Expressions)

## 📊 Dashboard Features  

### 🔹 KPI Metrics  
- Total Innings: **619**  
- Total Runs: **27,673**  
- Batting Average: **49.19**  
- Strike Rate: **91.43**  

---

### 🔹 Filters  
- Year Range: **2008 – 2025**  
- Format Selection: **ODI | Test | T20I**  

---

### 🔹 Visualizations  
- Runs by Year (Bar Chart)  
- Boundaries Analysis (4s vs 6s)  
- Hundreds & Fifties Trend (Line Chart)  

---

## 🔍 Key Insights  
- Peak performance during **2016–2018**  
- Consistent performance across formats  
- Majority runs scored through **fours**  
- Strong balance of average and strike rate  

---

## 🚀 How to Run  
1. Clone the repository  
2. Open Power BI Desktop  
3. Load the `.pbix` file from the `dashboard` folder  
4. Use filters to explore insights
